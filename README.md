# <Japanese Sake Meguri>

## サイト概要
### サイトテーマ
日本にある地酒や地ビール、果実酒、ワインなどのレビュー投稿をできるレビューサイト

### テーマを選んだ理由
もともとお酒が好きで、旅行に行った際にその土地の地ビールなどをよく飲むのですが、
何のお酒を飲んだか、どんな味のお酒だったのか覚えておらず、記録しておきたいと考え、このテーマにしました。

### ターゲットユーザ
20歳以上のお酒好きの人

### 主な利用シーン
旅行に行く時や、いつもと違ったお酒を飲みたい時、その土地の地酒などを知りたい時
